# DSA motion compensation

Проект содержит исследовательский прототип компенсации движения для цифровой субтракционной ангиографии.

Основная схема:

```text
mask, live
-> ROI и сосудистые маски
-> фоновые контрольные точки
-> локальный matching
-> non-rigid поле смещений
-> деформация маски
-> live - warped_mask
```

## Что в себя включает

- Морфологически усиленная `vessel_support_mask`.
- Исключение сосудов из выбора и сопоставления контрольных точек.
- Предпочтение non-rigid регистрации при достаточном числе надежных точек.
- Проверки matching: взвешенная ZNCC, градиентная схожесть, forward-backward последовательность, пространственная согласованность.
- Защита сосудистой области от яркостной коррекции.
- Мягкое подавление низкочастотной фоновой дымки после субтракции.
- Компактные метрики качества для фона и сосудистого сигнала.

## Запуск

```powershell
python run_v2.py --mask 0_coronary_mask.tiff --live 0_coronary_live.tiff --output-dir results\0_coronary
```

Для пакетного сравнения:

```powershell
python run_ablation.py --output-root results\ablation
```

## Сохраняемые результаты

В каждой папке результата остаются только необходимые файлы:

- `dsa_no_compensation.png` - субтракция без компенсации.
- `dsa_compensated.png` - итог после компенсации.
- `dsa_vessel_enhanced.png` - визуальное усиление сосудов после субтракции.
- `vessel_support_overlay.png` - широкая сосудистая маска.
- `displacement_magnitude.png` - модуль поля смещений.
- `match_vectors.png` - принятые и отклоненные контрольные точки.
- `motion_report.json` - компактный численный отчет.
- `summary.csv` - основные метрики.

## Ключевые модули

- `run_v2.py` - запуск одной пары изображений.
- `run_ablation.py` - сравнение рабочих вариантов.
- `dsa_v2/config.py` - параметры алгоритма.
- `dsa_v2/masks.py` - ROI, сосудистые маски, веса регистрации.
- `dsa_v2/features.py` - выбор контрольных точек.
- `dsa_v2/matching.py` - локальное сопоставление патчей.
- `dsa_v2/motion.py` - выбор модели движения.
- `dsa_v2/field.py` - построение и применение поля деформации.
- `dsa_v2/subtraction.py` - субтракция, яркостная коррекция, метрики.
- `dsa_v2/pipeline.py` - полный алгоритм и сохранение результатов.
